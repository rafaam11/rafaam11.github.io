Overview

하악골(턱뼈) 골절 수술 시 의사는 골편을 원래 위치로 복원한 뒤 금속판으로 고정해야 한다. 그러나 수술 전 복원 위치를 3D 영상으로 미리 계획하는 도구가 없어, 수술실에서 경험에 의존한 수작업으로 결정된다.

본 연구에서는 치아 교합(치아가 맞물리는 기하학적 관계)을 구속 조건으로 활용하는 최적화 알고리즘을 설계하여, 수술 전 CT 데이터만으로 골절 복원 위치를 자동 계산하는 Python 기반 시뮬레이터를 개발했다.
